# Legacy VHH Framework YAMLs

This folder holds VHH framework records that are intentionally not loaded by
the framework registry. They were moved out of the active panel because no
exact public RCSB coordinate-backed structure was identified for the selected
disease-targeting VHH domain, or because the reviewed coordinate entry did not
match the fixed framework segments in the YAML template.

These files are retained for provenance and possible future re-curation only.
Do not assume they are suitable for template-coordinate injection until an
exact framework-coordinate match has been found and validated.
