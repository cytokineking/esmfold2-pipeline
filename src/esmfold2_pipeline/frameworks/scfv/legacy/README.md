# Legacy scFv Framework YAMLs

This folder holds sequence-only scFv framework records that are intentionally
not loaded by the framework registry. They were moved out of the active panel
because no exact public RCSB coordinate-backed structure was identified for the
human or humanized YAML record.

These files are retained for provenance and possible future re-curation only.
Do not assume they are suitable for template-coordinate injection until an
exact framework-coordinate match has been found and validated.
